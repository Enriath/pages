QWC 3.2!

But what is it?

QWC is the Quoteless Word Counter, built for English Students.

How does it work?

Simply copy and paste your essay from the word processing software into the text box, and click Count.
If you check the Auto box, and click Start, it'll do it automatically!
The Settings (the cog) contains useful options, such as font size, font, text and background colour.

How do I get it?

Got Windows?

Get the .exe here: QWC-3.2.exe
If you can't get .exe files, get the .zip here: QWC-3.2.zip
Once extracted, run QWC.exe, or make a shortcut!

Got Apple OSX (Mac) or Linux?

Linux users need Python 2.7
Download this .zip and run QWC.py: QWC-3.2-OSX.zip
