QWC 3.2!

But what is it?

QWC is the Quoteless Word Counter, built for English Students.


How does it work?

Simply copy and paste your essay from the word processing software into the text box, and click Count If you check the Auto box, and click Start, it'll do it automatically!


How do I get it?

Got windows? Get the .exe here: QWC-3.2.exe If you can't get .exe files, get the .zip here: QWC-3.2.zip Once extracted, run QWC.exe, or make a shortcut!
Got Apple OSX (Mac) or Linux?
Make sure you have Python 2.7 (Macs do)
Download this .zip and run QWC.py: QWC-3.2-OSX.zip
